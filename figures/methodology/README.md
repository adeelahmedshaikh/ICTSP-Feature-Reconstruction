# Methodology Figures

This directory contains the methodology and conceptual figures used in the thesis:

**“Enhancing In-Context Time-Series Forecasting Through Feature Reconstruction”**

## Contents

The figures in this directory include:

- ICTSP forecasting pipeline diagrams
- Tokenization illustrations
- Conceptual framework diagrams
- Feature reconstruction visualizations

- Preprocessing integration workflow diagrams

## Purpose

These visualizations are used to explain:
- the forecasting workflow
- contextual token generation
- preprocessing integration
- feature reconstruction behavior
- experimental design

## Notes

The figures were prepared for thesis documentation and academic presentation purposes.
