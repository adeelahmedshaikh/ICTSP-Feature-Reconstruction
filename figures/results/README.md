# Experimental Result Figures

This directory contains the forecasting result visualizations and performance comparison figures generated during the ICTSP feature reconstruction experiments.

## Experiment Types

The figures include results from:

- Full-data forecasting
- Few-shot forecasting
- Zero-shot transfer forecasting

## Reconstruction Methods

The reported results compare:

- Baseline ICTSP
- STL Reconstruction
- Wavelet Reconstruction
- Singular Spectrum Analysis (SSA)

## Datasets

The figures may include experiments on:

- ETTh1
- ETTh2
- ETTm1
- ETTm2
- Exchange
- Weather

## Contents

This directory may contain:

- MAE comparison plots
- MSE comparison plots
- forecasting performance charts
- zero-shot transfer visualizations
- comparative experiment figures

## Notes

All result figures were generated using identical ICTSP configurations to ensure fair comparison between preprocessing methods.
