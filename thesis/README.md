# Master's Thesis

This directory contains the master's thesis document associated with the ICTSP feature reconstruction project.

## Thesis Title

**Enhancing In-Context Time-Series Forecasting Through Feature Reconstruction**

## Description

The thesis investigates the impact of reconstruction-based preprocessing methods on the forecasting performance of the In-Context Time-Series Predictor (ICTSP).

The study evaluates preprocessing approaches including:

- STL Decomposition
- Wavelet Reconstruction
- Singular Spectrum Analysis (SSA)

under multiple forecasting settings:
- full-data forecasting
- few-shot forecasting
- zero-shot transfer forecasting
